# Cleaned-Up

A clean user interface theme for [Obsidian](https://obsidian.md), forked from [Minimal](https://github.com/kepano/obsidian-minimal) by [@kepano](https://github.com/kepano).

## Features

- **JetBrains Mono** as the default font for editor, interface, and monospace
- Refined heading hierarchy with balanced weights and small-caps for H5/H6
- Wide readable line width (65% max)
- Clean spacing for headings and paragraphs
- Properties section moved to bottom in reading view

## Installation

1. Open Obsidian **Settings** → **Appearance**
2. Under **Themes**, click **Manage** and search for **Cleaned-Up**
3. Click **Install and use**

Or manually: clone this repo into your vault's `.obsidian/themes/` folder and select it in Appearance settings.

## Requirements

- Obsidian 1.9.0+
- [JetBrains Mono](https://www.jetbrains.com/lp/mono/) (recommended — falls back to system fonts)
